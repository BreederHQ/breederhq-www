# Glossary

Terms breeders and buyers use, with BreederHQ-specific meanings where relevant.

| Term | Meaning |
|---|---|
| Program | The breeder's overall breeding operation (species-agnostic). |
| Operation | Synonym for program, especially for livestock breeders. |
| Herd | The breeder's animals under management (also for camelids, sheep, cattle). |
| Kennel | The breeder's operation (dog-specific). |
| Cattery | The breeder's operation (cat-specific). |
| COI | Coefficient of Inbreeding, calculated automatically from pedigree with risk levels (LOW, MODERATE, HIGH, CRITICAL). |
| Pedigree | Multi-generation lineage tree, visualized with configurable depth and common-ancestor tracking. |
| GAID | Global Animal ID — cross-tenant animal linking. Enables real provenance across the BreederHQ network. |
| Exchange Code | The mechanism by which a stud owner authorizes another breeder to link a stud as sire on their offspring record. |
| Scout | The BreederHQ AI assistant. Agentic, data-backed, natural-language queries against the breeder's live program data. Includes Scout Genetics Scanner and Scout receipt-to-expense. |
| Help Assistant | A separate AI assistant for contextual guidance and searchable help. Distinct from Scout. |
| Rule of 7s | Standard early-puppy socialization protocol (dog-cluster). |
| ENS | Early Neurological Stimulation. Standard neonatal protocol (dog-cluster). |
| ESI | Early Scent Introduction. Standard neonatal protocol (dog-cluster). |
| Puppy Culture | Popular structured rearing protocol (dog-cluster). |
| Avidog | Alternative structured rearing protocol (dog-cluster). |
| Client portal | The buyer-facing surface where a specific buyer sees their animal, contracts, invoices, and messages. |
| Marketplace | The public discovery surface at /services/* and program/animal listings at marketplace.breederhq.com. |
| Embed widget | A snippet the breeder pastes into their own website (Squarespace, Wix, WordPress, Shopify, GoDaddy, HTML) that renders live BreederHQ marketplace listings from BreederHQ as the source of truth. |
| Profile Indicators | Account-history signals (Quick Responder, Established, Top Rated, Accepts Payments). Facts about platform behavior; not endorsements; cannot be purchased. |
| Stripe Identity passthrough | Display of a Stripe Identity verification status. BreederHQ does not verify; Stripe does. |
| Public-registry passthrough | Display of a public-registry status (OFA, AKC, AFA, etc.) where the issuing body maintains a public registry and BreederHQ has built the integration. |
| Self-attestation | Breeder or provider making a claim about themselves. Labelled as self-attested; not audited by BreederHQ. |
| Offspring | Species-neutral term for newborns (puppies, kittens, foals, kids, lambs, calves, crias, kits). Used on shared species-neutral pages. |
| Birthing | Species-neutral term for whelping, foaling, kidding, lambing, calving, queening, kindling. Used on shared species-neutral pages. |
| Trust page | The Identity and Credentials policy page at /trust. Legally reviewed. |
| FOUNDER50 | Promo code for 50% off Pro or Enterprise for 12 months. First 100 subscribers only. One redemption per customer. |
| Pro | Middle subscription tier ($79/mo). Includes Scout AI, Help Assistant, Genetics Lab, Auto-replies. |
| Enterprise | Top subscription tier ($159/mo). Everything in Pro plus team management, multi-location, automated follow-ups, unlimited marketplace listings. |
| Standard | Entry subscription tier ($29/mo). Full program-management foundation without Scout AI or Genetics Lab. |
